1. Px - Pixels are fixed value
  2. uh/vw - viewport height, viewport width
  3. % - percentage
  4. em - 1em (element) always equals to 16px (by default)
  5. rem - 1rem(reletive element) always equals to 16px (by default)

Pixels: Fixed value dosen't depend upon anything

vh/vw: Are depends upon viewport size

%: Are depends upon parent element properties

em: depends upon current element font-size by defaults 1em=16px

rem: depends upon current element font-size by defaults 1rem=16px
